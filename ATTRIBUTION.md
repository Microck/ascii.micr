# Attribution

## ASCII Art Generator

This project is a local web interface for generating high-quality ASCII art using gradient descent optimization.

## Original Projects

### gradscii-art
- **Author:** Zellic (stong)
- **Repository:** https://github.com/stong/gradscii-art
- **License:** GNU Affero General Public License v3.0 (AGPL-3.0)
- **Description:** An extremely good ASCII art generator based on machine learning using PyTorch and gradient descent.

### ascii.micr
- **Author:** Microck
- **Repository:** https://github.com/Microck/ascii.micr
- **License:** MIT License
- **Description:** Instant WebGPU ASCII art generator with gradient descent optimization and Y2K aesthetic.

## Fonts

This project uses the following fonts from the original gradscii-art project:
- `bitArray-A2.ttf` - Receipt printer font
- `gg mono.ttf` - Discord/coding font
- `SourceCodePro-VariableFont_wght.ttf` - Source Code Pro variable font
