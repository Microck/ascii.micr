# Grad-CAM ASCII Frontend

A Y2K-aesthetic frontend for the Grad-CAM ASCII generator.

## Setup

1. Install dependencies:
   ```bash
   bun install
   ```

2. Run development server:
   ```bash
   bun dev
   ```

3. Ensure the backend API is running on `http://localhost:8000`.

## Features
- **Themes**: Vapor, Cyber, Matrix, Hacker
- **Real-time**: Connects to Python backend for generation.
